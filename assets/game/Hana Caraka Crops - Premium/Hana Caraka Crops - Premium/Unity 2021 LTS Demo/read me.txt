
Tutorial
1. Copy 2d-extras-master to [project name]\Packages if you haven't import it 
   via package manager
   
2. Import Field.unitypackage

3. Create a tilemap [2D Object -> Tilemap -> Rectangular]

4. Open Tile Palette [Window -> 2D -> Tile Palette]

5. Select a tile, use a brush, and draw the map as you want to make it.
